# Progetto di algoritmi e principi dell'informatica 2020/21
Il progetto di quest'anno riguardava la lettura da file di matrici di adiacenza di vari grafi e la classificazione dei k migliori in base al loro cammino minimo, ovvero la somma dei cammini minimi dal nodo 0 al tutti i vari nodi del grafo. Il calcolo dei cammini minimi viene effettuato con l'algoritmo di Dijkstra(complessit√† O(|E| + |V|log|V|) ), mentre la classifica dei grafi viene mantenuta efficientemente con l'uso di un Max Heap.
